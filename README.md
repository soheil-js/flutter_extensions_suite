## Included Extensions
- [Dart](https://marketplace.visualstudio.com/items?itemName=Dart-Code.dart-code) - Dart language support and debugger for Visual Studio Code

- [Flutter](https://marketplace.visualstudio.com/items?itemName=Dart-Code.flutter) - Flutter support and debugger for Visual Studio Code

- [PUB STUDIO](https://marketplace.visualstudio.com/items?itemName=Mastersam.pub-studio) - Manage your flutter/dart packages directly from the editor. Quickly find, install, update, or remove packages without touching the command line. Perform other flutter related command without touching the command line

- [Flutter Checker](https://marketplace.visualstudio.com/items?itemName=Abdelkhalik.flutter-checker) - Check for outdated packages in Flutter/Dart pubspec.yaml files

- [Flutter Pubspec Links](https://marketplace.visualstudio.com/items?itemName=djbkwon.flutter-dependency-docs) - Float pub.dev links above dependencies in pubspec.yaml

- [Toggle Bracket Guides](https://marketplace.visualstudio.com/items?itemName=TGOSystems.togglebracketguides) - Easy way to toggle the bracket guides

- [Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments) - Improve your code commenting by annotating with alert, informational, TODOs, and more

- [Flutter Color](https://marketplace.visualstudio.com/items?itemName=circlecodesolution.ccs-flutter-color) - This plugin help you to visualize the colors used in your project

- [J2O - Json to Dart (freezed)](https://marketplace.visualstudio.com/items?itemName=BalaDhruv.j20) - Json to Dart classes converter including freezed classes

- [ADB Interface for VSCode](https://marketplace.visualstudio.com/items?itemName=vinicioslc.adb-interface-vscode) - An interface that allows ADB commands inside IDE with no need to have terminal access

- [Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens) - Improve highlighting of errors, warnings and other language diagnostics

- [IntelliJ IDEA Keybindings](https://marketplace.visualstudio.com/items?itemName=k--kato.intellij-idea-keybindings) - Port of IntelliJ IDEA Keybindings, including for WebStorm, PyCharm, PHP Storm, etc
